ITdirect auto-refresh for ticket searches for Google Chrome

Installation

Drag this bookmarklet into your Google Chrome bookmark bar (show with Ctrl+Shift+B): ITd Timer

Open and click the bookmarklet when you are in Home or Inbox.
You can enter the refresh time (5 mins is default and should be sufficient) and decide if you want to get desktop notifications.

How does it work

The script will simulate the user clicking on the "Go" button. It will than wait 5 seconds for the search to complete and try counting all lines.

It is designed to ignore lines containing a processor, as well as the lines containing [W],[w],[T] or [t].

Requirements:
ITdirect Language -> English
Columns: Description, Processor (optional: Priority (Description), Category, System ID)

Warnings

As the script is only simulating user-input blindly it can get confused. But the developers have taken every reasonable measure known to no cause problems.

This script is also neither developed nor endorsed by the ITdirect developers, and not guaranteed to be in sync with their developments. Expect it to break.

Known limitiations:

  • The script won't hit the Go button if a ticket is displayed/opened. Please use another browser windows for actually working on tickets.
  • As not all tickets are shown at once in paged result lists ITDtimer does not support them - please use scrolling instead.
  • If the search takes longer than 5 seconds the icon will not get updated. If this value hits your search please check if you can reduce the search time or give us a shoutout, so we can bump the waiting time.

Troubleshooting:

If the search lasts longer than 5 seconds please try reducing your search's complexity. You can also try executing the first search manually before activating ITDtimer.

The 5 seconds have been proven as enough for our team, if it is not usable for you please notify us, so we can adjust this upwards.

If ITDtimer stops working please try reloading the page and restart the script.
